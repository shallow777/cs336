# CS336 Assignments

This repository contains my work for Stanford CS336.

## Overview

The repo is organized by assignment, with each directory keeping its own code, tests, and assignment-specific documentation.

- [`assignment1-basics`](./assignment1-basics): my Assignment 1 work on core language model components and training fundamentals. This part is incomplete; I skipped the BPE portion.
- [`assignment2-systems`](./assignment2-systems): the Assignment 2 systems repository, focused on optimized attention, distributed training, and optimizer sharding. It currently contains the starter code and test layout as the base for implementation.

## Status

This repository is being used as a working course repo rather than a polished release artifact. Some assignments may be partially complete, under active development, or kept close to the original course starter structure.
